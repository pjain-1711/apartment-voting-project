Upload Nominees from Excel

Upload Excel File
Accepted formats: .xlsx, .xls
Required Columns:
  • Name - Nominee's full name
  • Apartment number - Flat/apartment number
  • Wing - Wing name (A, B, C, etc.)
  • Gender - "male" or "female"
  • Active status - Any value except "withdrawn"
Important Notes:
  • Nominees with Active status = "withdrawn" will be skipped
  • Duplicate nominees (same name + flat + wing) will be skipped
  • Wing must already exist in the system
  • Gender must be exactly "male" or "female" (case insensitive)
Excel Format Example

Your Excel file should look like this:

Name Apartment number Wing Gender Active status
John Doe 1001 A male active
Jane Smith 2305 B female active
Bob Wilson 1205 A male withdrawn
Rows with gray background will be skipped
Tips:
  • Optional columns (Timestamp, Email) will be ignored
  • Phone numbers are not collected from Excel
  • Make sure wings exist before uploading
  • Check for errors after upload

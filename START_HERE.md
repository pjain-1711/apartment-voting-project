## Quick Commands to Run the Application

```bash
# 1. Navigate to project
cd /Users/priyankajain/claude-learning/voting-system

# 2. Activate virtual environment
source venv/bin/activate

# 3. Run the application
python run.py
```

## 🎯 Initial Setup Steps

1. Login to admin: http://localhost:5000/admin/login
2. Add wings: "Manage Wings" → Add (A, B, C, etc.)
3. Add nominees: "Manage Nominees" → Add details
4. Enable voting: "Settings" → Toggle ON
5. Share URL with residents: http://localhost:5000
